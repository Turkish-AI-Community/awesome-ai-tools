# 🔍 Arama & RAG

> Vektör veritabanları, semantik arama ve Retrieval-Augmented Generation araçları.

---

## Araçlar

### Pinecone

- 🔗 **Link:** [pinecone.io](https://pinecone.io)
- 📝 **Açıklama:** Yönetilen, ölçeklenebilir vektör veritabanı platformu. RAG uygulamaları için popüler seçenek.
- 💰 **Fiyatlandırma:** Freemium
- 🌍 **Türkçe Desteği:** Yok
- ⭐ **Topluluk Notu:** 5/5
- 🧪 **Test Eden:** [Uğurcan Uzunkaya](https://www.linkedin.com/in/ugurcanuzunkaya/)
- 💬 **Kullanım Notu:** Ücretsiz katmanda 1 index ve 100K vektör destekliyor. Serverless mimarisiyle hızlı sorgu sunuyor.

---

### Chroma

- 🔗 **Link:** [trychroma.com](https://trychroma.com)
- 📝 **Açıklama:** Python ile kolayca entegre edilebilen, açık kaynaklı yerleşik vektör veritabanı.
- 💰 **Fiyatlandırma:** Ücretsiz (açık kaynak)
- 🌍 **Türkçe Desteği:** Yok
- ⭐ **Topluluk Notu:** 5/5
- 🧪 **Test Eden:** [Uğurcan Uzunkaya](https://www.linkedin.com/in/ugurcanuzunkaya/)
- 💬 **Kullanım Notu:** Prototipleme ve küçük-orta ölçekli projeler için ideal. LangChain ile sorunsuz çalışıyor.

---

### Qdrant

- 🔗 **Link:** [qdrant.tech](https://qdrant.tech)
- 📝 **Açıklama:** Rust ile yazılmış, yüksek performanslı açık kaynaklı vektör arama motoru.
- 💰 **Fiyatlandırma:** Freemium
- 🌍 **Türkçe Desteği:** Yok
- ⭐ **Topluluk Notu:** 4/5
- 🧪 **Test Eden:** [Uğurcan Uzunkaya](https://www.linkedin.com/in/ugurcanuzunkaya/)
- 💬 **Kullanım Notu:** Yerel Docker ile veya bulut hizmetiyle kullanılabiliyor. Filtreleme yetenekleri güçlü.

---

### Weaviate

- 🔗 **Link:** [weaviate.io](https://weaviate.io)
- 📝 **Açıklama:** GraphQL ve REST API destekli, çok modlu vektör arama platformu.
- 💰 **Fiyatlandırma:** Freemium
- 🌍 **Türkçe Desteği:** Yok
- ⭐ **Topluluk Notu:** 4/5
- 🧪 **Test Eden:** [Uğurcan Uzunkaya](https://www.linkedin.com/in/ugurcanuzunkaya/)
- 💬 **Kullanım Notu:** Hem vektör hem de anahtar kelime aramasını hibrid olarak destekliyor. Geniş ekosistemi var.

---

### Perplexity AI

- 🔗 **Link:** [perplexity.ai](https://perplexity.ai)
- 📝 **Açıklama:** Gerçek zamanlı web araması ile LLM yanıtlarını birleştiren AI arama motoru.
- 💰 **Fiyatlandırma:** Freemium
- 🌍 **Türkçe Desteği:** Var
- ⭐ **Topluluk Notu:** 5/5
- 🧪 **Test Eden:** [Uğurcan Uzunkaya](https://www.linkedin.com/in/ugurcanuzunkaya/)
- 💬 **Kullanım Notu:** Kaynaklara atıf yaparak yanıt üretiyor. Güncel bilgiye ihtiyaç duyulan sorular için vazgeçilmez.
